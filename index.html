DCS Compendium

This is a project created using C# and .NET in order to create a compendium for users who play the game "Digital Combat Simulator". It utilizes features such as updating, deleting entries, and searching for entries in a database aswell as retrieving information from a database. It has a user friendly UI with button click features for ease of access. I primarily made this so that less experienced players could have a quick way of accessing vital info during gameplay. During development, there was a problem with trying to return pictures for entries on other computers, eventually, we find out a good fix was to have it so that it creates the image again, but in the root so that it could access it.
